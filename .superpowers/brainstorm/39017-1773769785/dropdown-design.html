Timezone Picker Design

How should the compact From/To timezone selectors work? Click to choose.

A

Inline Search Overlay

Clicking the From/To pill opens a search overlay that replaces the pill area. Type to search — results filter live. Shows: ★ Recent (3) at top, then grouped results. Pressing Escape or clicking away closes it. Compact — fits within the extension width.

B

Dropdown Below Pill

Traditional dropdown that opens below the pill. Search input at top of dropdown. Scrollable list with continent groups. Similar to current but more compact — shorter timezone names (just city + offset), no long timezone name line. Max height constrained.

C

Full-Width Modal

Clicking pill opens a full-width search panel that covers the converter section. Large search input, generous spacing, easy to scan. Closes after selection. Feels like a proper search experience instead of a cramped dropdown.

Search must match:

  • City names — "Mumbai", "Calcutta", "NYC", "LA"
  • Country names — "India", "Japan", "USA"
  • Abbreviations — "EST", "IST", "CET"
  • IANA identifiers — "Asia/Kolkata"
  • Offsets — "+5:30", "UTC-5"
